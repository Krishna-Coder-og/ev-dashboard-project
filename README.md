# Electric Vehicle Market Analysis Dashboard

## Overview
This project presents a data-driven analysis of the Electric Vehicle (EV) market using Tableau for data visualization and Flask for web integration. The dashboard provides insights into EV model distribution, performance metrics, driving range, efficiency, and the relationship between vehicle price and range.

The objective of this project is to demonstrate how data analytics and visualization tools can be used to understand trends in the electric vehicle industry and support informed decision-making.

---

## Objectives

The main objectives of this project are:

- Analyze electric vehicle market trends across different brands and models.
- Visualize EV performance metrics such as speed, efficiency, and driving range.
- Explore the relationship between vehicle price and driving range.
- Provide an interactive dashboard that enables filtering and exploration of EV data.
- Integrate the Tableau dashboard into a web interface using Flask.

---

## Technologies Used

- Tableau Public – Data visualization and dashboard creation
- Python – Backend programming
- Flask – Web framework for dashboard integration
- HTML – Frontend interface
- GitHub – Version control and project hosting

---

## Dataset

The dataset used in this project contains information about electric vehicles including:

- Brand
- Model
- Range (Km)
- Top Speed (Km/H)
- Efficiency (Wh/Km)
- Price (Euro)
- Acceleration
- Segment
- Body Style

The dataset was cleaned and prepared before visualization.

---

## Data Preparation

The following steps were performed during data preparation:

1. Data cleaning and removal of inconsistencies.
2. Handling missing or incomplete values.
3. Creating calculated fields such as Cost per KM.
4. Structuring the dataset for visualization in Tableau.

---

## Visualizations

The dashboard contains several visualizations to analyze the EV market:

1. EV Models by Brand  
   Displays the distribution of EV models across different manufacturers.

2. Top Speed by Model  
   Highlights performance comparison among EV models.

3. Average Range by Brand  
   Shows average driving range offered by different brands.

4. Efficiency by Brand  
   Compares energy efficiency across EV manufacturers.

5. Price vs Range Relationship  
   Scatter plot illustrating the correlation between price and driving range.

6. EV Distribution by Segment  
   Shows how EV models are distributed across market segments.

---

## Dashboard Features

- Interactive filtering by brand and driving range.
- Multiple comparative charts for performance analysis.
- Data-driven insights for EV market trends.
- Clean and responsive dashboard layout.

---

## Story Analysis

The Tableau Story feature presents the analysis in three stages:

1. EV Market Overview  
   Provides a high-level view of EV models and brand distribution.

2. EV Performance Analysis  
   Focuses on metrics such as speed, efficiency, and range.

3. Price vs Range Insights  
   Explores the relationship between vehicle price and driving range.

---

## Web Integration

The Tableau dashboard is integrated into a web application using Flask. The dashboard is embedded using the Tableau Public embed API.
